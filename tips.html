Bash Commands

  • ls

    ls lists the items in a directory -- files or other directorys

    ls -a -l
  • touch

    touch creates a file in the specified directory

    touch hello.txt
  • mkdir

    similar to touch but makes a directory over a file

    mkdir directoryName

Git Commands

  1. add

    add adds a change in the working directory

    git add filename.txt
  2. commit

    commit commits the changes to the repository

    git commit -m "Implemented code"
  3. push

    push pushes the changed files to the repository

    git push

HTML Tags

  • <a>

    a links to another site

    <a> This links to another page </a>
  • <h1>

    h1 creates a header element; header has predefined size

    <h1> This is a header </h1>
  • <p>

    p aligns text in a paragraph

    <p> This is a paragraph </p>

CSS Concepts

  • tag {property: value;}

    This is an integral part of css, choosing which tag to stylize and how

    a {color: blue;}

    This modifies all a tags to be the color blue

  • Styles inherit from their parent

    If you want a style to be passed on to a child, css allows inheritance

    <a><h1>Header Link</h1></a> a {color: white;}

    This example code makes the header tag the color due to the anchor tag being styled as white
